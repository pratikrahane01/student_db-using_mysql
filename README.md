# student_db-using_mysql
A beginner-friendly SQL project showing how to convert CSV data into a structured MySQL database with tables, keys, and joins.

This project demonstrates how to convert CSV data into a structured MySQL database using MySQL Workbench. It covers database creation, table design, primary and foreign keys, data insertion, and different types of SQL joins.

It is a beginner-friendly project to understand real-world database concepts and practice SQL queries with hands-on examples.

#Features

CSV to MySQL data handling
Table creation with constraints
Use of PRIMARY KEY, UNIQUE, and FOREIGN KEY
CRUD operations
INEER, LEFT, RIGHT joins
Clean and simple SQL queries

Perfect for students starting their journey in SQL and backend development.
